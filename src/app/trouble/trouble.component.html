Troubleshooting

Becca won't connect!

Make sure you have properly set up the .env file to contain a DISCORD_TOKEN variable set to your token.

I can't get Becca in my server

Make sure you are generating the invite link correctly, and that you have "Manage Server" permissions for the server you are trying to add Becca to.

Becca isn't responding to a command!

Make sure Becca has the appropriate permissions in the server. To verify if a permission error exists, try granting her the Administrator permission. If the command starts working, then your permissions are mis-configured.

Other Issues?

If you are having another issue, you can create a bug report or in our Discord server.
